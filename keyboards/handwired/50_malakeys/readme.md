# 50_malakeys

Just other cheap jj_50 modified clone.

Keycaps used in this project: [DSA Style Keycaps by junaling - thingiverse](https://www.thingiverse.com/thing:4108829)

## Flashing

`make handwired/50_malakeys:default:avrdude`

When asked to, short the arduino pins RST to GND quickly and it should enter into bootloader mode and start to write.
